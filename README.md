# Login System

A simple Login and Registration application built using **Python**, **Tkinter**, and **MySQL**.
* User Registration
* User Login
* MySQL Database Integration
* Input Validation

## Requirements

* Python 3
* MySQL
* `mysql-connector-python`

Run the project:

```bash
python login.py
```

**Note:** Create the `login` database and `login` table before running the application.
